
# Links

Vite + React + TypeScript で構築されたツールのリンク集です。

## 概要

- Viteによる高速な開発環境
- React 19 + TypeScript
- HMR（ホットリロード）対応
- ESLint/Prettierによるコード整形・静的解析

## セットアップ

```bash
npm install
```

## 開発サーバー起動

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## コード整形・Lint

```bash
npm run format
npm run lint
```

## ディレクトリ構成

- `src/` ... アプリ本体
  - `App.tsx` ... メインコンポーネント
  - `assets/` ... 画像等のアセット
- `public/` ... 公開用静的ファイル
